# Arabic Discord RPG Bot Backend Snapshot

This is a backend-only project snapshot generated by `export_backend_snapshot.py`.

What is included:
- project structure
- Discord bot bootstrap
- settings and logging
- SQLite database initialization
- repositories and services
- story / jobs / quests / shop / achievements / weekly contracts
- faction reputation and NPC bonds
- admin / profile / world / story commands
- JSON content folders with scaffold files ready to be replaced

What is intentionally left as scaffold content:
- all production JSON content files

## Quick start

1. Create a virtual environment.
2. Install requirements.
3. Copy `.env.example` to `.env` and fill it.
4. Replace scaffold JSON files with your real content.
5. Run `python -m src.main`.
